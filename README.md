# FAQ Chatbot

## Overview

This project implements a simple FAQ chatbot that answers user questions based on a predefined set of frequently asked questions.
The chatbot matches user input with stored questions and returns the most relevant response.

## Features

* Simple chatbot interface
* Matches user questions with stored FAQs
* Displays chatbot responses instantly
* Easy to expand with more questions and answers

## Technologies Used

* HTML
* CSS
* JavaScript

## How It Works

1. The chatbot stores a list of frequently asked questions and answers.
2. The user enters a question in the chat input field.
3. The program checks the input against the stored questions.
4. If a match is found, the chatbot displays the corresponding answer.
5. If no match is found, the chatbot returns a default response.

## How to Run

1. Download or clone the repository.
2. Open **index.html** in any web browser.
3. Type a question in the chat input and press **Send**.
